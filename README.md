# Demi Results — Единая CRM

Объединённая CRM, собранная на базе бизнес-CRM (Next.js + Supabase) с
добавлением модулей AI-бота (диалоги, настройки, аналитика, отчёты),
работающих через FastAPI-бэкенд.

## Разделы

**Бизнес (Supabase):** Дашборд · Записи · Декомпозиция · Зарплата · Финансы · Маркетинг · Сотрудники · Настройки

**AI Бот (FastAPI):** Диалоги · Аналитика бота · Отчёты бота · Настройки бота

## Стек

- Next.js 16, React 19, TypeScript
- Tailwind CSS 4 + shadcn/ui
- Supabase (Postgres + Auth) — бизнес-модули
- FastAPI + Celery + Redis (внешний бэкенд) — модули бота

## Запуск локально

```bash
cp .env.example .env.local   # заполнить ключи
npm install
npm run dev                  # http://localhost:3000
```
